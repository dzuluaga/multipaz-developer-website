# Welcome to Multipaz Codelabs

Welcome to the Multipaz Codelabs section! Here you'll find hands-on, step-by-step guides to help you understand and implement key concepts in digital credential management using Multipaz.

Whether you're a developer, architect, or product owner, these codelabs are designed to give you practical experience and confidence in building secure, interoperable digital identity solutions with Multipaz.

---

## 💳 [DPC Lifecycle Codelab](/codelabs/DPC%20Lifecycle%20Codelab/)

Explore the full **Digital Payment Credential** lifecycle using a Read & Run model. Learn how a DPC is issued, stored in the wallet, and verified through a five-check precheck — all with existing SDK code.

| Module | What you'll learn |
|--------|------------------|
| 📋 Before You Begin | Prerequisites, tools, and codelab overview |
| 🏦 Issuance | How the issuer mints a DPC via OpenID4VCI |
| 👛 Wallet | How the wallet provisions, stores, and displays credentials |
| ✅ Verification | How the verifier runs 5 policy checks for payment eligibility |
| 🚀 Next Steps | Troubleshooting tips and further resources |
